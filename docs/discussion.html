Deux questions, une histoire coherente sur le cout de l’atypisme professionnel

Ce qu’on a trouve

Ce projet a pose deux questions sur les individus dont le genre est atypique pour leur profession. Les reponses dessinent une image coherente mais nuancee.

Question 1 -- Conditions de travail

Ce n'est pas le desalignement individuel qui coute -- c'est la feminisation du metier qui concentre la precarite. Les femmes en metier feminin cumulent temps partiel subi, sous-emploi et faible anciennete. Les hommes dans ces memes metiers subissent le temps partiel mais pas le reste.

La devalorisation est structurelle, pas individuelle.

Question 2 -- Mobilite professionnelle

Le desalignement structure la mobilite de facon asymetrique. La femme en metier masculin dans le prive change plus souvent de metier. L'homme en metier feminin dans le public quitte plus souvent l'emploi. Dans les deux cas, l'atypisme genere de l'instabilite -- mais dans des contextes tres differents.

Le cout de l'atypisme est sectoriel et genere.

L’histoire commune

Les deux analyses racontent la meme histoire sous deux angles :

La feminisation d’un metier concentre la precarite – temps partiel, sous-emploi, rotation elevee. C’est une caracteristique structurelle des secteurs, pas une penalite individuelle pour atypisme.

Mais l’atypisme individuel a ses propres effets – sur la mobilite. La femme qui entre dans un metier masculin (surtout dans le prive) y est moins stable. L’homme qui entre dans un metier feminin (surtout dans le public) en repart plus souvent. Ces effets sont separes de la devalorisation structurelle.

Deux theories, deux contributions

Par rapport a la litterature sur la devalorisation (Levanon et al., 2009 ; Goldin, 2014) : on confirme que la feminisation d’un metier degrade les conditions de travail – mais on montre que cet effet touche aussi les hommes dans ces metiers, pas seulement les femmes.

Par rapport a la theorie d’Akerlof et Kranton (2000) : le cout de deviation identitaire existe dans le domaine professionnel, mais il se manifeste sur la mobilite (instabilite), pas sur les conditions de travail au sens strict. C’est coherent avec les resultats du projet precedent sur les loisirs – le cout est comportemental, pas psychologique.

Les limites

Limite majeure

Pas de mesure d'identite individuelle

Le desalignement est mesure par l'ecart entre sexe declare et norme professionnelle -- pas par une mesure d'identite continue. On ne peut pas distinguer une femme a identite tres feminine dans un metier masculin d'une femme a identite androgyne.

Extension naturelle : apparier avec une mesure d'identite externe (score LASSO de l'EPC via imputation par age, diplome, secteur).

Limite importante

Selection dans les professions

Les individus ne sont pas assignes aleatoirement a leur profession. Les femmes qui choisissent des metiers masculins different probablement sur des dimensions non observees. L'effet estime mele desalignement et selection.

Limite secondaire

Absence de salaire

L'outcome naturel pour tester la devalorisation est le salaire -- absent de l'Enquete Emploi de base. Le module EECSA permettrait de completer l'analyse.

Perspectives

01

Ajouter le salaire

Le module EECSA (Enquete Emploi Complement Salaires) permettrait de tester directement la devalorisation salariale -- l'outcome le plus documente dans la litterature.

02

Integrer l'identite continue

Apparier avec une mesure d'identite individuelle (score LASSO de l'EPC) permettrait de distinguer l'effet du desalignement de genre de l'effet de la feminisation du secteur.

03

Dynamiques temporelles

Les vagues successives de l'Enquete Emploi permettraient de tester si la devalorisation precede ou suit la feminisation d'un metier -- une question causale centrale dans la litterature.

04

Hors marche du travail

Combiner avec les resultats du projet sur les loisirs culturels : les individus desalignes professionnellement le sont-ils aussi dans leurs pratiques culturelles -- et ces deux formes de desalignement s'additionnent-elles ?
